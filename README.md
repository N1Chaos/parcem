# PARCEM — Commentaire d’écoute musicale

PARCEM est une plateforme pédagogique interactive développée dans le cadre des travaux du GiPTiC – Académie de Paris.

Elle permet aux élèves de réaliser un **commentaire d’écoute structuré**, écrit et oral, à partir d’une œuvre musicale.

🔗 Accès en ligne : https://parcem-9fdebf.forge.apps.education.fr/index.html

---

## 🎯 Objectifs pédagogiques

PARCEM vise à :

- Développer l’analyse musicale par l’écoute active
- Enrichir le vocabulaire musical
- Structurer un commentaire écrit argumenté
- Situer une œuvre dans l’espace et dans le temps
- Encourager l’autoévaluation
- Travailler l’expression orale

---

## 🎵 Fonctionnalités principales

### 🎧 Analyse sonore
- Import de fichiers audio
- Lecteur multimédia intégré (Plyr)
- Égaliseur (basses / médiums / aigus)
- Balance stéréo
- Spectre fréquentiel
- Spectrogramme
- VU-mètres gauche / droite
- Formes d’onde

### 🧠 Aide à l’analyse musicale
- Vocabulaire musical classé par catégories :
  - Caractéristiques principales
  - Paramètres musicaux
  - Structure et style
- Sélection guidée de termes
- Carte du monde interactive (Leaflet)
- Frise chronologique

### ✍️ Rédaction assistée
- Éditeur de texte enrichi
- Connecteurs logiques organisés par fonction
- Mise en forme (gras, italique, listes)

### 📊 Autoévaluation
- Grille critériée sur 20
- Organisation
- Vocabulaire
- Identification des phénomènes sonores
- Situation spatio-temporelle

### 🎙️ Expression orale
- Enregistrement audio intégré
- Téléchargement du commentaire

### 📄 Export
- Export en fichier Word
- Génération de fichier texte
